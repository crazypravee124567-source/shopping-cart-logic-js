Shopping Cart

A simple Shopping Cart web project built with plain HTML, CSS, and JavaScript. It demonstrates core JavaScript array methods to manage cart items and calculate the total price.



 Live Demo

> Open `index.html` directly in your browser — no setup needed!



  Project Structure


shopping-cart/
├── index.html    # Main file (HTML + CSS + JS combined)
└── README.md     # Project documentation
```



  Features

- ✅ Display cart items in a clean table layout
- ✅ Add new products to the cart
- ✅ Remove products from the cart
- ✅ Update product quantity
- ✅ Calculate and display **Total Amount**
- ✅ Fully responsive and clean UI



  JavaScript Concepts Used

| Concept | Used For |
|--------|----------|
| `Array.push()` | Add new product to cart |
| `Array.filter()` | Remove a product from cart |
| `Array.forEach()` | Update quantity & render table rows |
| `Array.reduce()` | Calculate total price |
| `DOM Manipulation` | Display cart data on the page |
